# THE MESS

This is my personal game that I am making. The game is not good in any way, but that is not the point. The point of this game is for me to have fun developing stuff, rather than having a playable final product.

It has stolen assets, spaghetti code, bugs, unfinished parts, and a general lack of polish.

## Backstory

This is a continuation of an old RPG Maker game by the same name. I stopped using that and switched to Godot because I found RPG Maker to be too limiting. Almost nothing from the original is in this, except for a single level, but the reason for it's exsitance remains the same.
